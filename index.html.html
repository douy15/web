Ảnh Nội dung Liên hệ

Bài tập môn Xây dựng ứng dụng web — sinh viên Đỗ Đại Uy. Trang này được tạo để thực hành HTML ngữ nghĩa, danh sách, ảnh và liên kết bên ngoài.

Nội dung chính

  • Thẻ ngữ nghĩa cơ bản: header, main, footer
  • Thư viện ảnh với mô tả bằng thuộc tính alt
  • Liên kết đến trang tài liệu bên ngoài (MDN Web Docs)

Tài liệu tham khảo

Nếu bạn muốn tìm hiểu thêm về HTML và các chuẩn web, có thể xem tại MDN Web Docs. Đây là nguồn học tập rất hữu ích cho sinh viên CNTT.
